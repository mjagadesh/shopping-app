# shopping-app
this Django Project describes the CURD(create,update,read,delete) operation of the shopping


Steps to run the DRF:

step 1:
    change the database setting in  setting.py 
step 2:
    execute the python manage.py makemigrations command 
step 3:
    execute the python manage.py migrate command 
step 4:
    run the dev server through python manage.py runserver command
